## 🌟 Premium Eid Greeting Card Generator

An exquisite, highly interactive web application designed to create, personalize, and download high-resolution Eid greeting cards. Choose from beautiful curated templates, type your name, select custom typography, and drag elements around to design the perfect holiday wishes!

### ✨ Key Features
- **4 Premium Art Templates:**
  - *Royal Night:* Deep indigo and glittering gold skies with hanging lanterns and a glowing crescent moon.
  - *Emerald Grace:* Rich green backdrop with an intricate golden geometric mandala.
  - *Sunset Gold:* Warm sunset sky gradient silhouettes mosque minarets and sparkling particles.
  - *Pastel Rose:* Minimalist modern rose aesthetic featuring clean line-art details.
- **Dynamic Personalization:** Input custom recipient names or family signatures.
- **Interactive Drag & Drop:** Click/tap and drag text overlays directly on the canvas preview to position them anywhere.
- **High-Resolution Export:** Downloads cards as perfect `1200x1200px` PNG images—optimized for Instagram, WhatsApp, and social media sharing.
- **Zero-Dependency Pure Stack:** Built with pure HTML5, CSS3, and Vanilla JavaScript (runs instantly on any browser with no package installs!).

## 🚀 How to Run Locally

You can launch and use the application immediately on your computer:
1. Open the repository folder on your computer.
2. Double-click the [index.html](index.html) file to open it in your web browser.
3. Start personalizing and generating cards!

Alternatively, you can run a lightweight development server using Node:
```bash
# Run a quick local server (if Node/NPM is installed)
npx http-server .
```

## 🌐 Deploy to GitHub Pages (Host for Free!)

To share this web application live with your friends and family, host it using **GitHub Pages** for free:

1. **Commit and Push changes to GitHub:**
   ```bash
   git add .
   git commit -m "Add premium Eid Greeting Card Generator"
   git push origin main
   ```
2. **Enable GitHub Pages:**
   - Go to your repository on GitHub.
   - Click on the **Settings** tab.
   - On the left sidebar, click **Pages** (under the "Code and automation" section).
   - Under **Build and deployment**, set the source to **Deploy from a branch**.
   - Under **Branch**, select `main` (or `master`) and folder `/ (root)`.
   - Click **Save**.
3. **Enjoy your live site!**
   - GitHub will generate a URL like `https://<your-username>.github.io/github-badges-helper/` where your greeting card generator is now hosted live for anyone to use!
